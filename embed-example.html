Sunsynchro Calculator — Embed Example

This page shows how to embed the calculator and automatically resize the iframe using postMessage.

Host page script (listening for height)

<script>
function receiveMessage(e){
  // If you host calculator.html on a different origin, replace '*' with the calculator origin and verify e.origin
  if(!e.data || typeof e.data.sunsynchroHeight === 'undefined') return;
  const iframe = document.querySelector('iframe[data-sunsynchro]');
  if(iframe) iframe.style.height = e.data.sunsynchroHeight + 'px';
}
window.addEventListener('message', receiveMessage, false);
</script>

That's it — when the calculator loads or changes size it will post its height to this host page and the iframe will resize.
